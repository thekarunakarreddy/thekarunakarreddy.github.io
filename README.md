## Getting Started

This project follows a structured SDLC approach with comprehensive documentation in the `docs/` folder.

## Development Phases

1. **Planning & Requirements** - Define project scope and requirements
2. **System Design** - Architecture and technical specifications
3. **Implementation** - Code development
4. **Testing** - Quality assurance and validation
5. **Deployment** - GitHub Pages deployment
6. **Maintenance** - Ongoing updates and improvements

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- GitHub Pages (Hosting)
